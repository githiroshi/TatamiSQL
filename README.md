# TatamiSQL

TatamiSQL is a database client designed for databases running in local development environments. 

<img width="3024" height="2224" alt="スクリーンショット 2026-07-06 20 47 08" src="https://github.com/user-attachments/assets/90077d67-cf58-4e0b-aa50-a5df1a22f9bb" />

MySQL is currently supported.

Download the latest version from [TatamiSQL GitHub Releases](https://github.com/githiroshi/TatamiSQL/releases).

TatamiSQL is intended for local development, not production databases. Use it at your own risk.

## Opening TatamiSQL on macOS

The macOS app is ad-hoc signed, but it is not signed with an Apple Developer ID or notarized by Apple. macOS therefore displays a warning when the app is opened for the first time.

Only continue if you downloaded the DMG from the official [TatamiSQL GitHub Releases](https://github.com/githiroshi/TatamiSQL/releases) page.

1. Open the DMG and drag `TatamiSQL.app` into the Applications folder.
2. Open `TatamiSQL.app`, then close the warning dialog displayed by macOS.
3. Open **System Settings > Privacy & Security**.
4. Scroll down to the Security section and click **Open Anyway** for TatamiSQL.
5. Authenticate if prompted, then click **Open** in the confirmation dialog.

Subsequent launches can be performed normally.

## macOSでTatamiSQLを開く方法

Apple Developer IDによる署名とAppleの公証は未実施です。そのため、初回起動時にmacOSの警告が表示されます。

公式[TatamiSQL GitHub Releases](https://github.com/githiroshi/TatamiSQL/releases)からDMGをダウンロードした場合のみ、以下の手順を実行してください。

1. DMGを開き、`TatamiSQL.app`を「アプリケーション」フォルダへドラッグする。
2. `TatamiSQL.app`を開き、macOSの警告ダイアログを閉じる。
3. **システム設定 > プライバシーとセキュリティ**を開く。
4. 「セキュリティ」までスクロールし、TatamiSQLの**このまま開く**をクリックする。
5. 必要に応じて認証し、確認ダイアログの**開く**をクリックする。

次回以降は通常どおり起動できます。
